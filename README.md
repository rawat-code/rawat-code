## 🧠 About Me

I'm a student who spends most of his free time building things and entering hackathons — I like the pressure of a deadline and the challenge of shipping something real in a short window. Most of my projects sit at the intersection of **AI and everyday productivity** — to-do apps, deadline trackers, assistants — things I'd actually want to use myself.

I'm not chasing buzzwords; I just enjoy the process of taking an idea, wiring up the frontend, backend, and an AI piece that actually does something useful, and getting it in front of people to try. Design matters to me as much as function — I want what I build to feel good to use, not just work.

Still learning a lot, still figuring things out, and always up for the next hackathon.

## 🚀 Featured Projects

<details>
<summary><b>🎯 Deadline Zero — AI-Powered Task & Deadline Management Platform</b></summary>
<br/>

An AI-integrated productivity platform designed to help users manage tasks and deadlines intelligently, combining a modern animation-rich interface with real backend logic for deadline prioritization and tracking.

| Attribute | Detail |
|---|---|
| **Stack** | Next.js, React, Tailwind CSS, Node.js, Vercel |
| **Scale** | Multi-user task management with real-time state |
| **Performance** | Optimized rendering with animated, glassmorphic UI components |
| **Security** | Environment-based config, secure API routes |
| **Impact** | Hackathon submission focused on demo-readiness and UX polish |
| **Repository** | [github.com/rawat-code/deadline-zero](https://github.com/rawat-code/deadline-zero) |

Built as a hackathon submission, this project emphasizes both engineering completeness and visual craftsmanship — featuring a neural-network-inspired canvas background, particle effects, and a fully overhauled glassmorphic design system layered on top of a functional deadline-tracking engine.

</details>

<details>
<summary><b>⚡ The Last-Minute Life Saver — AI Productivity Companion</b></summary>

A single-file, instantly deployable AI productivity companion combining task management, habit tracking, a calendar view, and analytics — built for rapid hackathon deployment via Netlify Drop.

| Attribute | Detail |
|---|---|
| **Stack** | HTML, CSS, Vanilla JavaScript |
| **Scale** | Single-file, zero-dependency architecture |
| **Performance** | Lightweight, instant load, no backend latency |
| **Security** | Fully client-side, no external data exposure |
| **Impact** | Built for the Vibe2Code hackathon under tight time constraints |
| **Repository** | [github.com/rawat-code/last-minute-life-saver](https://github.com/rawat-code/last-minute-life-saver) |

Designed as a rapid-build hackathon entry, this project demonstrates the ability to ship a complete, multi-feature productivity tool as a single deployable file — including a simulated AI assistant driven by lightweight keyword-matching logic.

</details>

| **Scale** | Modular plugin-based architecture |
| **Performance** | Real-time speech-to-text and low-latency voice response |
| **Security** | Local SQLite memory store, sandboxed system commands |
| **Impact** | Personal AI assistant with browser and system automation capability |

A comprehensive personal-assistant system integrating speech interfaces, LLM providers (Anthropic, OpenAI, Ollama), and automation tooling into a single cohesive desktop application with a plugin-based extensibility model.
